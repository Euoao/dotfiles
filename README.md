# dotfiles
@Euoao does dotfiles


## Tool set
Please ensure these tools are installed before setup:
- [oh-my-zsh](http://ohmyz.sh/)
- [tmux](https://github.com/tmux/tmux) ( >= 3.2)

## Usage

### Full Installation
```bash
./install.sh
```

### Configuration Update Only
If you only want to update configurations:
```bash
./config_setup.sh
```
